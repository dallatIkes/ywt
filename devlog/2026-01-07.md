## What I learned

- To create a new page, you have to add `{NAME_OF_YOUR_PAGE}/+page.svelte` file in the `routes` folder
- You can add shared components (like navbar) in the `+layout.svelte` file at the root of the folder you want the `+page.svelte` file and all of its child to share
